Access rights

Description of the accessibility to the dataset files.

The dataset files may be made accessible to users and/or re-users in various ways. The following categories are available:

  • Open access

    The dataset files are accessible to all users of EASY and 'CC0 Waiver - No Rights Reserved' applies. For more information please visit https://creativecommons.org/about/cc0. In this category all possible rights (such as copyrights and database rights) on the dataset files have been waived.

  • Open access for registered users

    The dataset files are accessible to all registered users of EASY.

  • Restricted: archaeology group. (Only available for the discipline archaeology)

    The dataset files are only accessible to archaeologists registered at EASY.

  • Restricted: request permission.

    The dataset files are only accessible for registered users of EASY who have been granted permission to access the dataset files. You as depositor have the exclusive ability to grant or deny a permission request. A notification of a permission request is sent to the Email address used for your user account

  • Other access

    It is not possible to access the dataset files through EASY. You can only choose this option if the dataset files are accessible, for example, through a different data repository, digital archive, databank or website. Please note: a specific agreement with DANS needs to be negotiated.

    Regardless of the applicable category, users should always cite datasets.

    NB: The descriptive information on the research project and the dataset (metadata) is always freely accessible to all users of the Internet. This provides each potential user or re-user the opportunity to find the dataset files and to assess whether they are valuable to his/her research.
