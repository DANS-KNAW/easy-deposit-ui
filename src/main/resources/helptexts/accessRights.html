The DANS policy is "Open Access when possible and Restricted Access when necessary".

  1. If your data or metadata contain personal data, the dataset falls under "Restricted Access". You as depositor have the exclusive ability to grant or deny a permission request. A permission request is sent to the email address used for your user account. The DANS licence, which automatically applies to a Restricted Access dataset, describes what a future dataset user can and should do after being granted permission.
  2. In specific cases you might decide that access to a dataset without personal data must still be restricted. When you choose this option, the permission request process and the DANS licence apply as described above. When in doubt, please contact DANS at info@dans.knaw.nl.
  3. However, if your dataset falls under "Open Access", you must select either the Creative Commons Zero Waiver (CC0) or a Creative Commons licence.

Note that the metadata you are now providing are always freely accessible to everyone, even when access to the data is restricted.
