The field 'Content type' can be used to indicate the form of the dataset in general terms: a technical description of the contents. It is recommended that you do this in conformity with the international thesaurus called 'DCMI resource type'. The default value is 'Dataset' for a collection of different files belonging to one research project. If the files of the dataset are only documents (PDF or text files), the dataset should be typed as 'Text'. If the files of the dataset are only image files, the dataset can be typed as 'Image' and so on.

Please use the free text field only if the pull-down menu is not applicable.
