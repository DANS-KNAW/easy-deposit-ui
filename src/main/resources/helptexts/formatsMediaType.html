Format

Software used.

This field can be used to indicate which software and which file format was used to store the data. Indicate which software was used, and preferably also fill in the name of the manufacturer, the product name, and the version

Click the button behind a field to add an additional field.

Examples

  • 90-77287-02-6 (ISBN)
  • 1836 (Archis Onderzoeksnummer)
  • 52189 (Archis Waarnemingsnummer)
  • NHDA: R0214
  • 0925-6229 (RAAP projectnummer)
  • STAR: P1233
