General characteristic of the dataset deposited.

The field ‘Type' can be used to indicate the form of the dataset in general terms. You can opt for both an archaeological description of the contents and a technical description of the contents. If you opt for a technical description, it is recommended that you do this in conformity with the international thesaurus called ‘DCMI resource type'. The default value will then be ‘Dataset'.

Click the button behind a field to add an additional field.
