You can use this field to send information or requests to the DANS data manager. For example, in case the paper to which the dataset relates has not yet been published, you can mention in this field where and when the paper is expected to be published.

If your files are too big to upload, you can mention that here, and submit the dataset without the files. The data manager who processes your dataset will contact you to discuss alternative methods of depositing and accessing the files.

If you want to deposit a dataset with both "Open access" and "Restricted access" files, please contact DANS first via info@dans.knaw.nl. During depositing you use this field to indicate which files should be made available according to which access category. The dataset itself will be of category "Restricted Access".
